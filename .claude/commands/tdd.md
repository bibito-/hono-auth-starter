# /tdd コマンド

引数で指定した機能・ファイルを TDD（Red → Green → Refactor）で実装する。

## 実行手順

1. [.claude/steering/current.md](../steering/current.md) を確認する
   - 進行中タスクあり → ゴールと「次のステップ」から再開する
   - なし → `steering/current.md` に新規エントリを作成してから開始する
2. [.claude/skills/tdd-workflow.md](../skills/tdd-workflow.md) を読む
3. 引数の対象に対してワークフローを実行する

対象: $ARGUMENTS
